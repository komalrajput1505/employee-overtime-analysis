# Employee Overtime Pattern Study 📊

This project analyzes employee working hours and overtime patterns.

## 🔍 Objectives:
- Identify overtime trends
- Analyze department workload
- Improve productivity

## 📊 Techniques Used:
- Data Analysis using Pandas
- Data Visualization using Seaborn and Matplotlib

## 📈 Output:
- Overtime distribution graph
- Hours vs overtime relationship

## 🛠 Tools:
Python, Pandas, Seaborn, Matplotlib
